**Decisions made:**

- **Frontend: React SPA + FastAPI** — single Render service, single deployment. Vite builds to `frontend/dist/`, FastAPI serves it. No Next.js.
- **Standalone repo** — ScriptRipper is not a fork of the original; it's a ground-up rewrite. Fresh repo, not fork. MANUS_Document_Repository is a document/asset repo; code projects live independently.
- **Idempotency constraint** — `(subscription_id, source_url)` unique, not `source_url` globally. Same URL in two subscriptions is allowed.
- **Service stubs are real implementations** — `llm.py`, `transcription.py`, `email.py`, `storage.py` are fully written, not placeholder stubs. Will be verified end-to-end in Phase 2.
- **Alembic migration deferred** — held until next session. User must have `DATABASE_URL_EXTERNAL` set locally before running.
